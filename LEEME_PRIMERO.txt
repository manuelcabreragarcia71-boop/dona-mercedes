🚀 INICIO RÁPIDO (3 PASOS)

1. CONFIGURAR FIREBASE
   → Ve a https://supabase.com/
   → Crea un proyecto "dona-mercedes"
   → Habilita Supabase Postgres y Storage
   → Copia credenciales a supabase.config.js

2. INSTALAR Y EJECUTAR
   → cd dona-mercedes
   → npm install
   → npm start

3. PROBAR EN TU MÓVIL
   → Descarga "Expo Go" desde Play Store
   → Escanea el QR que aparece
   → ¡La app se abrirá!
